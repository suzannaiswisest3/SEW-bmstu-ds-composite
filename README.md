# Composite Materials Prediction Web App
# Комплексный дипломный проект: Интеллектуальная система проектирования композитов

Данный репозиторий содержит полный цикл разработки выпускной квалификационной работы студента МГТУ им. Баумана — от исследовательских этапов обучения моделей до финального программного комплекса.

## Структура репозитория

1. Исследовательская часть (Хронология разработки в Google Colab):
* SEWBASE_ALLPIPELINES.ipynb — базовый ноутбук с процессами обучения моделей, сравнения пайплайнов и выбора лучших архитектур.
* SEW_DEVAPP (1).ipynb — исследовательский блокнот с первыми тестами и экспериментами по созданию веб-приложения.
* SEW_production.ipynb — рабочий ноутбук с финальными отлаженными скриптами запуска и подготовки расчетных алгоритмов.

2. Практическая часть (Программный комплекс веб-интерфейса):
* app.py — главный исполняемый файл графического веб-интерфейса Streamlit.
* requirements.txt — конфигурационный манифест внешних программных зависимостей для сборки веб-контейнера.
* Файлы моделей (*.pkl, *.keras) — сохраненные оригинальные веса обученных моделей машинного и глубокого обучения.
